Etiquetas HTML de Formato

Etiqueta de Énfasis

<em>: Permite enfatizar el texto.

Etiqueta de Marcado

<mark>: Permite marcar un texto.

Etiqueta de Encabezado

<h1>: Permite incluir encabezados en el documento.

Etiqueta para Indicar Importancia en el Texto

<strong>: Permite marcar un texto como importante.
